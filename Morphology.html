Morphology Module¶

Morphology.hitmiss(src, element)[source]¶

Morphological operation for shape detection. The notation for the structuring element uses 1 for foreground, -1 for background and 0 for ignore.

Parameters:
  • src – source image
  • element – structuring element for the desired shape as a square matrix
Returns:

the hit-or-miss transformation of the source image
Return type:

Image

The algorithm can be described as below:

A⊗B = (A⊖B1)∩(A⊖B2)
