Welcome to Angiogram Sprout Extraction’s documentation!¶

As a simple breakdown, the Driver intitiates a set of analysis tasks for testing. An analysis task consists of a sample image and produces a SproutAnalysis which is printed by the ShollAnalysisReport.

Contents:

  • Driver: initiates a set of analysis tasks
  • Extraction: extractors to extract features from an angiogram
  • Features: extractable features from an angiogram
  • Geometry: computational geometry objects
  • REPL: Read-Eval-Print-Loop
  • ReportGeneration: report generators
  • SetForest: data structure for the union-find algorithm
  • ShollAnalysis: quantitative analysis method for analyzing angiogram sprouts
  • SproutSegmentation: segmentation strategy for angiogram sprouts
  • SteerableFilter: adaptive filter for restoring angiogram sprouts
  • utils: utilities
